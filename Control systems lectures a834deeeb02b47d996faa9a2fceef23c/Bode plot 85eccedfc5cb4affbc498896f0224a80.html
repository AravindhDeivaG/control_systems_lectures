Bode plot

Refer to this script for bode plot matlab code

Used to represent phase and gain of a transfer function at steady state for an input of given frequency.

The fundamental building blocks of a transfer functions are

  • Constants
  • Zeros (Transfer function tends to zero)
  • Poles (Transfer function tends to infinity

We’ll look at bode plots of some of the following

  • Constant
  • Zero and pole at origin
  • Real zero and pole
  • Complex zero and pole
