Scope of FV theorem:

  • Let us divide the S plane into four regions
    • Right hand plane (To the right of imaginary axis) RHP
    • Left hand plane (To the left of imaginary axis) LHP
    • Imaginary axis
    • Origin
  • If the system has one or more pole in RHP or imaginary axis, them FV theorem is not applicable as the system either oscillates indefinitely (pole on imaginary axis) or the value approaches infinity (pole on RHP).
  • Hence FV theorem is applicable only when the poles are at origin and/or LHP

Steady state error:

  • A good control system follows the input as close as possible.
  • For instance the response of a system to ramp function should be nearly equivalent to a ramp function.
  • Since the FV of the ramp function tends to infinity, it doesn’t make sense to use FV as a parameter to evaluate the control system’s performance. Hence we add a new term called steady state error.
